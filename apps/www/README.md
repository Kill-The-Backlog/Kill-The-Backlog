# Kill The Backlog Marketing Site

Marketing landing page for Kill The Backlog.

## Development

```bash
# From workspace root
devbox run pnpm dev --filter=@ktb/www
```

Or from this directory:

```bash
devbox run pnpm dev
```

The site runs at `localhost:4321`.

## Build

```bash
devbox run pnpm build
```

Output is generated to `./dist/`.
